## Senior Backend Engineer

We are looking for a Senior Backend Engineer to design and build scalable Java and Spring Boot services. You will develop REST APIs in a microservices environment, work with PostgreSQL and Redis, and improve performance and reliability.

### Requirements

- 5+ years of Java and Spring Boot experience.
- Strong API design, database design, SQL, PostgreSQL, and query optimization skills.
- Experience with distributed systems, Docker, Kubernetes, CI/CD, and deployment automation.
- Knowledge of authentication, authorization, security, code review, mentoring, and Agile delivery.
- Ability to lead technical planning and communicate system-design decisions.
