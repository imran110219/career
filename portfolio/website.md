# Personal Website

Use this file to plan and maintain personal website content.

## Site Goals

| Goal | Notes |
| --- | --- |
| Establish credibility | Show role focus, technical depth, and professional judgment. |
| Support applications | Give recruiters and hiring managers a quick way to validate experience. |
| Showcase work | Link to projects, writing, GitHub, and contact channels. |

## Recommended Pages

- Home: concise positioning, key skills, selected work.
- Projects: detailed project case studies.
- Writing: articles and technical notes.
- Resume: downloadable PDF and summary.
- Contact: professional contact links.

## Checklist

- [ ] Website content matches current resume positioning.
- [ ] Links to GitHub, LinkedIn, and writing are current.
- [ ] Resume download is the latest approved PDF.
- [ ] Analytics and contact forms do not expose private data.
