# Publications

Track articles, technical writing, talks, papers, and public references.

## Publication Template

| Field | Notes |
| --- | --- |
| Title | Public title. |
| Type | Article, talk, paper, tutorial, podcast, or documentation. |
| Venue | Website, conference, publication, or organization. |
| Date | Publication or presentation date. |
| Link | Canonical public URL. |
| Summary | One-sentence explanation of topic and value. |
