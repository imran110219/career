## Usage Notes

| Resume Context | Recommendation |
| --- | --- |
| Senior technical roles | Keep education concise unless it is highly relevant. |
| Architecture roles | Include research, systems, or engineering depth if applicable. |
| Management roles | Include leadership, business, or organizational coursework if relevant. |
| International applications | Confirm degree naming, date format, and location conventions. |
