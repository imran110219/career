# AI Security

AI security is optional differentiation for these target roles. It should not delay Lead Backend, Senior Backend or Software Architect applications unless the target job description makes AI security central.

## Topics

- Prompt-injection defence.
- Sensitive-data leakage prevention.
- Model output validation.
- Guardrails and policy checks.
- LlamaGuard or similar safety classifiers.
- Retrieval access control.
- Data retention and audit logging.
- Human review requirements.
- Model and dependency supply-chain risk.

## Practice Tasks

- Threat model a support chatbot or AI-assisted log summarizer.
- Define controls for preventing confidential data from entering prompts.
- Create a review checklist for AI-generated code and architecture suggestions.
