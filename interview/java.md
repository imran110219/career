# Java Interview Notes

## Core Topics

| Topic | Review Focus |
| --- | --- |
| Collections | HashMap internals, ordering, complexity, concurrency implications. |
| Concurrency | Threads, executors, synchronization, locks, atomics, CompletableFuture. |
| JVM | Memory model, garbage collection, class loading, profiling basics. |
| Language | Generics, streams, records, sealed classes, exceptions, immutability. |
| Testing | JUnit, Mockito, integration testing, test data design. |

## Practice Prompts

- Explain how `HashMap` works and what changed after treeification of buckets.
- Compare `synchronized`, `ReentrantLock`, and atomic classes.
- Diagnose a memory leak in a long-running service.
- Design a thread-safe cache with expiration.
- Explain checked versus unchecked exceptions and when each is appropriate.
