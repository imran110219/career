# Interview Evidence Log

This file records evidence generated through learning, labs, personal projects, or professional work. It does not automatically promote a topic into [skills inventory](../master/skills-inventory.md) or a résumé.

## Evidence Labels

| Label | Meaning |
| --- | --- |
| Learning only | Study notes or conceptual practice; not implementation evidence. |
| Lab evidence | Implemented in a controlled lab or exercise. |
| Personal project evidence | Implemented in a public or private personal project. |
| Professional evidence | Used in employment or client work, with disclosure limits respected. |
| Needs verification | Claim exists but scope, metric, ownership, or disclosure is not confirmed. |
| Résumé-safe | Evidence is specific, defensible, and safe to include for the target role. |
