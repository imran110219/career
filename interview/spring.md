# Spring Interview Notes

## Core Topics

| Topic | Review Focus |
| --- | --- |
| Spring Boot | Auto-configuration, starters, profiles, configuration properties. |
| Dependency Injection | Bean lifecycle, scopes, constructor injection, circular dependencies. |
| Web APIs | Controllers, validation, exception handling, filters, interceptors. |
| Data | Spring Data JPA, transactions, migrations, repositories, query tuning. |
| Security | Authentication, authorization, JWT, method security, CORS, CSRF. |
| Operations | Actuator, health checks, metrics, structured logging. |

## Practice Prompts

- Explain how Spring Boot auto-configuration works.
- Design a consistent API error response strategy.
- Debug a transaction boundary issue.
- Secure an API used by both internal services and external clients.
- Explain when to use JPA, JDBC, or a custom query.

## Checklist

- [ ] Review transaction propagation and isolation.
- [ ] Prepare examples of validation and error handling.
- [ ] Review security configuration patterns.
