# Software Architect Mock Interviews

## 75-Minute Architect Mock

| Time | Segment |
| --- | --- |
| 10 min | Background and architecture scope. |
| 20 min | System design case. |
| 15 min | Non-functional requirements and trade-offs. |
| 15 min | Migration, integration or governance scenario. |
| 10 min | Security, cloud and operability. |
| 5 min | Candidate questions. |

## Practice Scenarios

- Define boundaries and data ownership for an election workflow platform.
- Design Open-Care for reliability, security and searchable resources.
- Create an ADR for modular monolith versus microservices.
- Plan disaster recovery assumptions for a high-stakes reporting workflow.
- Propose governance that improves consistency without blocking delivery.
