# Checklists

## Resume release

- [ ] Claims map to an inventory/evidence record.
- [ ] Current role uses present tense; completed work uses past tense.
- [ ] Metrics identify whether they are personal, team-level, reported, or verified.
- [ ] No private contact data appears in a public build.
- [ ] A4 PDF has selectable text, no blank pages, intact headings, and no split company heading.
- [ ] Markdown and generated outputs are synchronized.

## Application package

- [ ] Job URL, status, and next action are in `applications/tracker.csv`.
- [ ] Selected resume track matches the role.
- [ ] Private contacts, salary, and offer notes are in ignored paths.
- [ ] Interview stories use factual STAR prompts and respect confidentiality.
